
# AOV Builds – Full Skeleton (React Router + Tailwind)

Scaffold đa trang theo cấu trúc site gốc, *không* chứa nội dung/hình ảnh bản quyền.
Bạn có thể cắm dữ liệu của mình để hoàn thiện.

## Cài đặt
```bash
npm i
npm run dev
```

## Build
```bash
npm run build
npm run preview
```
